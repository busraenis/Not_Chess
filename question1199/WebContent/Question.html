Assignment 2:

Not Chess

Please read important information first.

Description

 In this project, you will create a game based on chess with easier rules.
 The game is played with chess pieces on the chessboard (8x8).
 You will print the chess board and take the next move as input until the game is over.
 Details are explained below.

 If you have any questions regarding the assignment, please send an email to cmpe150.mail@gmail.com and
 use Assignment-2 in subject.
 Reminder: Discussing the ways to solve the problems in the assignments with each other is allowed, but
 code sharing is strictly forbidden.

Game Board

 You are expected to print the board and pieces as shown below (left).

 

 Black pieces are shown as lowercase letters and white pieces are shown as uppercase letters. Each piece also has a value.
  
Letter Piece Value
P or p Pawn 1
R or r Rook 5
N or n Knight 3
B or b Bishop 3
Q or q Queen 9
K or k King 999

Game Rules

 Pieces move as they do in chess. You can find more information here but you don't need to learn them for this assignment.
 You can assume that the players will move the pieces correctly.
 White makes the first move.
 Each player can only make 1 move on their turn.

Inputs

 Player will enter the move as "piece_column piece_row new_column new_row" without the whitespaces. An example move can be "e2e4" which takes the pawn on "e2" and moves it to "e4" for the board above.

Expected Move Controls

 You are expected to check the cases below for each move
 After taking the move from player as an input
  #1- if the selected location ("e2" for move "e2e4") has no piece or the new location ("e4" for move "e2e4") has a same colored piece you should not accept the input
  #2- if enemy king is taken, the game is over
 You are NOT expected to code any other scenario. Remember that this is not chess, just a chess inspired game.

Outputs:

 You are expected to
  - Print the initial board (in Game Board section) before the first move.
  - Print the current board after each move.
  - Print "White's turn" or "Black's turn" after printing the board
  - Print "Invalid Move!" if there is an invalid move input (#1 in Expected Move Controls section) and take the input again.
  - Print the current score after each move (after printing the board, except for the first round). This score is calculated by adding the values of pieces for both players. White is positive and black is negative.
  - Print "GAME OVER!" and print the winner at the end as "WHITE WON!" or "BLACK WON!".

 Check the example to have a better understanding.

Keep in mind that we will be grading your code not just based on these examples, but other cases as well, so try to write code which can handle all possible cases.

Warning: You are not allowed to use any imports.
